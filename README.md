# Stacktrack

AI-powered precious metals and coin intelligence app — scan, value,
and track your stack from your phone.

## Features
- AI coin scanner (Claude Vision) — ID, grade, melt value, retail range
- Live spot price tracking (gold, silver, platinum)
- Portfolio tracker with cost basis and P&L
- Coin collection registry with photo storage
- Flip tracker for buy/sell tracking
- Single HTML file, zero dependencies, works offline once loaded

## Deploy
Open `index.html` in any modern browser, or host on GitHub Pages /
Netlify for mobile access via "Add to Home Screen."
